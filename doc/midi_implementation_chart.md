### バンクセレクト（CC#0 / CC#32）

対象は **MIDI ch 1–9, 11–16**（メロディチャンネル）のみ。**MIDI ch 10**（リズム）でも CC は受信するが、リズム音源はノート番号で音色を決めるため **バンク・PC は音響に影響しない**。

#### 通常の送り方

シーケンサは多くの場合、次の順で送る。

```
CC#0 (MSB) → CC#32 (LSB) → Program Change → Note On
```

| 段階 | 受信内容 | チャンネル状態 | 音が変わるタイミング |
|------|----------|----------------|----------------------|
| CC#0 | バンク MSB | 受信 MSB を保持（LSB 待ち） | **まだ変わらない** |
| CC#32 | バンク LSB | 適用バンクを決定（下記「バンク正規化」参照） | **まだ変わらない**（音色番号は維持） |
| PC | 音色番号 0–127 | 音色番号を更新（非 CSM 時は適用バンク 0:0） | **即時**（発音中の音色も変わる） |
| Note On | — | その時点の適用バンク・音色番号で発音 | 新規・再トリガに反映 |

#### バンク正規化

本機は GM Level 1 相当のメロディ音色セットを搭載する。非 GM バンク（XG / GS 等）は **音色選択上 GM バンク (0:0) に正規化** する。Program Change は **常に受け付け**、音色番号を GM セットに割り当てる（マップ外リズムノートと同様、エラーにはしない）。

| 受信バンク (MSB:LSB) | 適用バンク | 備考 |
|----------------------|------------|------|
| **0:0** | 0:0 | GM メロディ |
| **3:x** | 3:x | CSM 対応時は CSM モードへ切替 |
| 上記以外 | **0:0**（正規化） | 受信バンクは記録するが音響には GM バンクとして扱う。PC は GM 音色番号として適用 |

CSM バンク（MSB=3）を選ぶとメロディチャンネルは **CSM モード** になる（CSM 対応時）。GM バンク (0:0) または非 CSM バンクに戻すと通常のメロディ発音に戻る。

#### 音色セットについて

現状、メロディの FM 音色は **GM 128 音色** のみ。XG/GS 用の別音色セットは未搭載のため、非 GM バンクを受信しても **GM の同じ音色番号** で鳴る。

### RPN / NRPN（Data Entry MSB = CC#6）

MIDI ch 1–9, 11–16 でのみ **Data Entry MSB（CC#6）** が音響に効く。MIDI ch 10 では CC#98–101 による RPN/NRPN **選択値は保持**するが、Data Entry は無効（音響変化なし）。

**Data Entry は最後にアドレスした RPN（CC#100/#101）または NRPN（CC#98/#99）に適用される。** RPN をアドレスすると NRPN 選択は無効化され、逆も同様。

| 選択 | パラメータ | 動作 |
|------|-----------|------|
| RPN 0:0 | Pitch Bend Sensitivity | ピッチベンド感度（±半音）。変更時、発音中に即時反映 |
| RPN 0:2 | Coarse Tuning | 粗いチューニング（半音単位、中央値 64）。対応ビルド時のみ有効 |
| RPN 0:5 | Modulation Depth Range（GM2） | **未実装**。ビブラート深さは固定レンジ |
| NRPN 1:8 | Vibrato Rate（XG） | ビブラート速度。XG 準拠の 64 中心相対値（64=プリセットレート相当） |
| NRPN 1:9 | Vibrato Depth（XG） | ビブラート深さ。XG 準拠の 64 中心相対値（64 以下=深さ 0、65 以上を CC#1 相当の深さへスケール） |
| 上記以外 | — | 無視 |


### リズム (MIDI ch 10) の Note Off

| 操作 | 音響 |
|------|------|
| Note Off / Note On velocity=0 | **無効**（発音は YM2608 の減衰で終了。シンバルもチョークしない） |
| 排他ノートの Note On 切替（42↔44↔46 等） | 前のノートの楽器を **全チップで damp** |
| CC#120 / CC#123 | **全リズム楽器を damp** |

### チャンネル別の違い

| MIDI ch | 種別 | ピッチ系（PB / CC#1 / NRPN 1:8–9） | 備考 |
|---------|------|-------------------------------------|------|
| 1–9, 11–16 | メロディ | ◯ | ソフトウェア LFO ビブラート。RPN 0:5 は未実装 |
| 10 | リズム | × | YM2608 リズム 6 種。Note Off / vel=0 は無視（減衰任せ）。排他 HH 等は Note On 切替で damp。CC#120/#123 で全 damp |
| （共通） | CSM モード | × | CSM 発音時は PB / ビブラート非対応 |

- モード1: オムニ・オン、ポリ
- モード2: オムニ・オン、モノ
- モード3: オムニ・オフ、ポリ
- モード4: オムニ・オフ、モノ
- ◯：あり
- ×：なし

**チャートの見方については [MIDI 1.0 規格書](https://amei.or.jp/midistandardcommittee/MIDI1.0.pdf) の「MIDIインプリメンテーションチャートの作成法」を参照。**
